fastpagesのセットアップ方法と使い方

fastpages はipynbファイルをblogエントリとして公開することを支援してくれるツールです。 このblogもfastpagesを使って公開しています。 このエントリではfastpagesのセットアップと使い方を紹介します。

fastpagesのセットアップ方法

  1. https://github.com/fastai/fastpages/generate をクリックし、あなたが関わるどのgithubアカウントを使うかなどを設定し、fastpagesをテンプレートとしたプロジェクトを作ります。ただし {your-username}.github.io のプロジェクト名は使えません。
  2. 1.の作業の数秒後に 1.で作ったプロジェクト宛にpull requestが送られてきます。このpull requestのページを開いてください。そのpull requestには3.以降の操作のインストラクションが書かれています。
  3. https://8gwifi.org/sshfunctions.jsp を開き ssh の key-pair を作ります。RSA と 4096 を選択し Passphrase は空にします。
  4. https://github.com/あなたのgithubアカウント名/1.のプロジェクト名/settings/secrets に移動し Add a new secret をクリックします。3.のPrivate KeyをValueフィールドにコピペします。NameフィールドにはSSH_DEPLOY_KEYを入れます。
  5. https://github.com/あなたのgithubアカウント名/1.のプロジェクト名/settings/keys に移動し Add deploy key ボタンをクリックします。1.のPublic KeyをKeyボックスにコピペします。｀Titleにはfastpages-keyとでも名前付けしましょう。｀Allow write access にはチェックを入れてください。最後にAdd keyはクリックしてください。

  6. 5.が済んだら2.のpull requestをmergeしてください。

  7. が済んだらその数分後に https://あなたのgithubアカウント名.github.io/1.のプロジェクト名 にfastpagesのサイトができあがっています。

fastpagesの使い方

https://あなたのgithubアカウント名.github.io/1.のプロジェクト名/tree/master/_notebooks にipynbを追加するとブログエントリが出来上がります。 ただしYYYY-MM-DD-*.ipynbというルールでノートブックを名前付けする必要があります。 例えば下記のような名前です。

2020-01-28-My-First-Post.ipynb
2012-09-12-how-to-write-a-blog.ipynb
